Commands

Here are the most used commands which you should know

!command

(add|remove|list)
Either add a command, remove
a command, or list all the commands
which the bot knows

!bot

Tells who made the bot!

!reload

Reloads all the commands

!quote

(add|remove|list)
Either add a number at the end to
either run the quote or remove it
while list gives the amount of quotes

!reg

(add|remove|list)
Just like Nightbot, anyone who's
in the reg list will not get timed out

!word

(add|remove|list)
If you add a word or phrase anyone
who says that phrase will be timed out
unless you are a mod or a regular

!ping

Returns Pong!

!restart

Restarts the bot

Google Document

The bot requires you to have a google doc with a certain template.
Below you can click the button and select File > Make a Copy... to make a copy of your own.

A few things to note:


Not all commands need to start with '!'; they can start with anything.
Any command that starts with '$' can only be run by the streamer
Any command that starts with '&' can only be run by a mod
Any command without '$' or '&' can be run by anyone
